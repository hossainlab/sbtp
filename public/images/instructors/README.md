# Instructor Images

Place instructor profile images in this folder.

## File Naming Convention

Use lowercase names with hyphens:
- `jubayer-hossain.jpg`
- `muhib-shahjan.jpg`
- `muntasim-fuad.jpg`

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 400x400px (square)
- **File Size**: < 200KB (optimized for web)

## Usage

Reference images in `src/constants/index.ts`:

```typescript
image: "/images/instructors/jubayer-hossain.jpg"
```
